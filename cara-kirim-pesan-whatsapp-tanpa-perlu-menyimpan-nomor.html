Cara kirim pesan whatsApp tanpa perlu menyimpan nomor

WhatsApp adalah salah satu aplikasi perpesanan yang paling banyak digunakan di dunia. Bahkan, di banyak wilayah / negara, ia mempertahankan tempat nomor 1 (oleh pengguna). Jadi, ini adalah platform ideal bagi pelanggan untuk terhubung dengan bisnis dan juga bagi bisnis untuk terhubung dengan pelanggan mereka. Tetapi umumnya tidak cocok untuk terus menambahkan semua nomor telepon sebagai kontak. Mempertahankan pandangan ini, WhatsApp memungkinkan pengguna untuk memulai obrolan tanpa terlebih dahulu menyimpan nomor telepon sebagai kontak di perangkat. Keuntungan

Mengirim pesan WhatsApp tanpa menyimpan nomor memiliki beberapa keuntungan. Beberapa dari mereka terdaftar di bawah ini:
- Itu tidak mengacaukan daftar kontak Anda.
- Anda dapat memulai obrolan dari WhatsApp Web, tanpa harus membawa ponsel.
- Cepat dan mudah - menghemat waktu.

A. Melalui WhatsApp Web :
Jika Anda menggunakan WhatsApp Web pada PC Anda, maka Anda dapat memulai obrolan dengan nomor telepon dengan mengikuti langkah-langkah ini:
1. Pastikan Anda masuk ke WhatsApp Web. Buka web.whatsapp.com untuk mengonfirmasi.
2. Catat nomor telepon beserta kode negara, tetapi tanpa menambahkan "+" atau "00".
3. Misalnya, jika pengguna WhatsApp berasal dari Indonesia (+62) dan nomor teleponnya adalah 08880 98880.
4. Anda akan menggunakan: 62888098880 Tambahkan ke akhir teks berikut: https://web.whatsapp.com/send?phone=
5. Sebagai contoh: https://web.whatsapp.com/send?phone=62888098880
6. Rekatkan ke browser web Anda dan tekan Enter.
7. WhatsApp Web akan memuat dan kemudian membuka jendela obrolan untuk nomor telepon ini.

B. Melalui Aplikasi Android atau IOS
Anda juga dapat memulai obrolan dari ponsel Android atau iPhone. Sama sekali tidak perlu menyimpan nomor telepon. Cara ini bisa digunakan utuk semua versi WhatsApp, baik itu WA iPhone, WA original, maupun WA MOD Begini caranya:
1. Catat nomor telepon lengkap pengguna WhatsApp.
2. Anda juga memerlukan kode negara tanpa + awalan.
3. Misalnya 0888098880 Buka browser web di ponsel Anda.
4. Misalnya, Google Chrome di Android dan Safari di Apple.
5. Ketik URL berikut di browser web Anda diikuti dengan nomor telepon dari langkah pertama: wa.me/62888098880
6. Tekan tombol Enter atau tombol go.
7. Tautan akan teratasi, dan aplikasi WhatsApp akan terbuka secara otomatis dengan jendela obrolan.

Sekarang Anda dapat mulai mengirim pesan ke pengguna WhatsApp tanpa menyimpan nomor teleponnya di kontak.
